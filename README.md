# InvoQ - Invoice Generator

InvoQ is a streamlined invoice creation platform designed for independent professionals, startups, and service providers. Our tool simplifies the process of generating polished, personalized invoices, helping you maintain a professional image and improve your cash flow.
## Features

- User-friendly interface for creating invoices
- Step-by-step form for inputting invoice details
- Real-time preview of the invoice as you create it
- Responsive design for both desktop and mobile use
- Client-side form validation using react-hook-form


### Technologies Used

- **PDF Generator:** [React PDF](https://react-pdf.org/)
- **Framework:** [Next.js](https://nextjs.org)
- **Styling:** [Tailwind CSS](http://tailwindcss.com)

### Running Locally

Ensure that you have pnpm installed.

```bash
git clone git@github.com:Suraj370/InvoQ.git
cd invoQ
pnpm install
pnpm run dev
```

Create `.env.local` file similar to `.env.example`.
